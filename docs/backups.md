# Backups

Make backup from your applications in Discloud

* [backups](#backups)

```sh-session
USAGE
  $ discloud backups [APP_ID]

ALIAS
  $ discloud backup [APP_ID]
  $ discloud bkp [APP_ID]
  $ discloud b [APP_ID]

ARGUMENTS
  APP_ID The id of your app on Discloud
```

* If does not contains `APP_ID`, this get all of your apps.
