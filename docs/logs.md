# Logs

View the logs from application in Discloud.

* [logs](#logs)

```sh-session
USAGE
  $ discloud logs [APP_ID]

ALIAS
  $ discloud l [APP_ID]

OPTIONS
  -s, --save Write the log locally

ARGUMENTS
  APP_ID The id of your app on Discloud
```

* If does not contains `APP_ID`, this get all of your apps.
