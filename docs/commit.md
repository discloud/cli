# Commit

Commit one app or site to Discloud.

* [commit](#commit)

```sh-session
USAGE
  $ discloud commit PATH [APP_ID]

ALIAS
  $ discloud c PATH [APP_ID]

ARGUMENTS
  APP_ID The id of your app on Discloud
  PATH   The path of file name
```

* If contains `ID` in `discloud.config`, `APP_ID` is optional.
