# Comandos

## Start

Inicie um ou todos os seus aplicativos no Discloud.

- [start](#start)

```sh-session
USO
  $ discloud start [APP_ID]

ALIAS
  $ discloud i [APP_ID]

ARGUMENTOS
  APP_ID O id do seu app no ​​Discloud.
```

- Se não contiver `APP_ID`, a CLI mostrará uma lista de seus aplicativos para você selecionar.
