# Comandos

## Logs

Visualize os logs do aplicativo no Discloud.

- [logs](#logs)

```sh-session
USO
  $ discloud logs [APP_ID]

ALIAS
  $ discloud l [APP_ID]

OPÇÕES
  -s, --save Gravar o log localmente.

ARGUMENTOS
  APP_ID O id do seu app no ​​Discloud.
```

- Se não contiver `APP_ID`, obterá todos os seus aplicativos.
