# Comandos

## Stop

Interrompa um ou todos os seus aplicativos no Discloud.

- [stop](#stop)

```sh-session
USO
  $ discloud stop [APP_ID]

ALIAS
  $ discloud p [APP_ID]

ARGUMENTOS
  APP_ID O id do seu app no ​​Discloud.
```

- Se não contiver `APP_ID`, a CLI mostrará uma lista de seus aplicativos para você selecionar.
