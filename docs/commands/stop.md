# Commands

## Stop

Stop one or all of your apps on Discloud.

- [stop](#stop)

```sh-session
USAGE
  $ discloud stop [APP_ID]

ALIAS
  $ discloud p [APP_ID]

ARGUMENTS
  APP_ID The id of your app on Discloud
```

- If does not contains `APP_ID`, the CLI will show a list of your apps for you to select.
