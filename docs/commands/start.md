# Commands

## Start

Start one or all of your apps on Discloud.

- [start](#start)

```sh-session
USAGE
  $ discloud start [APP_ID]

ALIAS
  $ discloud i [APP_ID]

ARGUMENTS
  APP_ID The id of your app on Discloud
```

- If does not contains `APP_ID`, the CLI will show a list of your apps for you to select.
