# Commands

## Delete

Delete one or all of your apps on Discloud.

- [delete](#delete)

```sh-session
USAGE
  $ discloud delete [APP_ID]

ALIAS
  $ discloud del [APP_ID]
  $ discloud rb [APP_ID]

ARGUMENTS
  APP_ID The id of your app on Discloud
```

- If does not contains `APP_ID`, the CLI will show a list of your apps for you to select.
